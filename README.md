# Text Vector Visualizer

A simple tool that uses vector embeddings and dimensionality reduction to arrange texts in a 1D, 2D, and 3D positioning diagram. This is useful for comparing and positioning texts like brands, book summaries, song lyrics, etc.

## Setup and Usage

1.  **Install Docker and Docker Compose.**
    - [Docker](https://docs.docker.com/get-docker/)
    - [Docker Compose](https://docs.docker.com/compose/install/)
2.  **Run the application.**
    ```bash
    docker compose up --build -d
    ```
3.  **Access the application.**
    - Open your browser and navigate to `http://localhost`.
